<h1>Bill Tracker Back-End</h1>
<p>This project represents the back-end of the Bill Tracker Web-Based Application. This was created to allow users to securely log in using Passport.js, as well as add, update and keep track of their monthly outgoings all in one place.</p>
  
<p><strong>Tech-stack</strong></p>
<ul>
  <li><strong>Main Language:</strong> TypeScript</li>
  <li><strong>Framework:</strong> Express.js</li>
  <li><strong>Authentication:</strong> Passport.js, Passport-Local</li>
  <li><strong>Testing:</strong> Jest, Supertest, Supertest-Session</li>
  <li><strong>Database:</strong> MongoDB</li>
</ul>

<h1>Live-version:</h1>
<p>Front-End of project to be created in 2024.</p>

<h2>How to run and install the project:</h2>
<ol>
  <li>Fork and clone the project <code>git clone &lt;project link> </code></li>
  <li>Go to the projects folder in the Terminal <code>cd &lt;folder path> </code></li>
  <li>Run the command <code>npm install</code> to install all required dependencies</li>
  <li>Run the command <code>npm run dev</code> to allow all Typescript code inside the <code>/src</code> folder to be compiled into the <code>/dist</code> folder</li>
</ol>
